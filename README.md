# Extreme (Front-end)
날씨정보를 이용한 야외 액티비티 예약 애플리케이션

## 팀 구성
| 권낙현 | 정연휘 | 허다은 |
|---|---|---|
|[knh6269](https://github.com/knh6269)|[JungYeonHwi](https://github.com/JungYeonHwi)|[qor4](https://github.com/qor4)|


## 주요 기능
- 액티비티 상품 정보 제공
- 상품 상세 정보 제공(리뷰, 상품 상세 정보)
- 검색 기능 
- 날씨 정보를 이용한 일정 예약 기능
- 마이페이지(주문 내역)
- 날씨 지도(전국 날씨 정보 제공)

## Stack
- React
- HTML
- CSS
